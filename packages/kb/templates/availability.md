# 庫存詢問回覆模板（availability）

適用場景：用戶詢問庫存、現貨、尺寸、顏色是否有貨

---

## DM 模板

### availability_dm_01 — 確認庫存引導
```
您好！感謝詢問 😊 請告知您想了解的款式及尺寸，我將立即為您確認最新庫存狀況！
```

### availability_dm_02 — 補貨通知引導
```
您好！您詢問的款式目前暫時售完，預計補貨時間請加入 LINE 好友（@{LINE_ID}）以接收到貨通知，或持續關注我們的 IG 最新貼文 🙏
```

### availability_dm_03 — 預購引導
```
您好！感謝您的支持 ❤️ 部分熱門款式目前有開放預購，請告知您感興趣的款式，我們將說明預購方式及預計到貨時間！
```
