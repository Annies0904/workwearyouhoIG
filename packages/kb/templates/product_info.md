# 商品資訊詢問回覆模板（product_info）

適用場景：用戶詢問商品規格、材質、尺寸、使用方式等詳細資訊

---

## DM 模板

### product_info_dm_01 — 引導查看說明
```
您好！感謝您對我們商品的興趣 😊 詳細的商品規格請參考貼文說明，若有特定問題（材質、尺寸等）歡迎告知，我將詳細為您說明！
```

### product_info_dm_02 — 尺寸選擇建議
```
您好！為了幫您選到最合適的尺寸，請告知您的身高/體重，我將根據尺寸對照表為您推薦最適合的號碼 👗✨
```

### product_info_dm_03 — 材質說明
```
您好！我們的商品採用優質材質製作，具體材質成分請告知您想了解的款式，我將提供完整的材質說明 🌿
```
