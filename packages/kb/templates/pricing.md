# 價格詢問回覆模板（pricing）

適用場景：用戶詢問商品價格、費用或優惠

---

## DM 模板

### pricing_dm_01 — 通用價格引導
```
您好！感謝您的詢問 😊 我們的商品定價依款式而異，請告知您感興趣的款式，我將立即為您提供詳細報價！
```

### pricing_dm_02 — 引導至官方頁面
```
您好！目前完整的定價資訊請參考我們的官方頁面連結，若有特定款式想了解，歡迎直接告訴我，馬上為您確認最新售價 🙌
```

### pricing_dm_03 — 提及優惠活動
```
您好！很高興您對我們的商品感興趣 ❤️ 目前我們有不定期優惠活動，建議加入 LINE 好友（@{LINE_ID}）以掌握第一手優惠資訊，也歡迎告知有興趣的款式，我們將提供最新報價！
```
