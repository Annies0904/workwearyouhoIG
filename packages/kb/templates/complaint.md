# 客訴 / 退換貨回覆模板（complaint）

適用場景：用戶表達不滿、客訴、要求退換貨

---

## DM 模板

### complaint_dm_01 — 通用致歉引導
```
您好，非常抱歉造成您的困擾 🙏 請告知您的訂單編號及遇到的問題，我們將優先為您安排處理，務必讓您滿意！
```

### complaint_dm_02 — 商品瑕疵
```
您好，對於商品問題深感抱歉！煩請提供訂單編號及瑕疵部位的照片，我們收到後將立即安排換貨或退款處理，請放心 💪
```

### complaint_dm_03 — 配送問題
```
您好，對於配送延誤造成的不便深感抱歉 😔 請提供您的訂單編號，我們將立即查詢物流狀態並儘快為您解決！
```

### complaint_dm_04 — 退貨處理
```
您好！依消費者保護法，商品到貨 7 天內可申請退貨（需全新未使用）。請提供訂單編號，我們將為您說明退貨流程 🙏
```
